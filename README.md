### Prerequisites

What things you need to install the software :
* Get an OS
* Install flutter
* Android Studio / VSCode or any ide


### Running

Run this in terminal
```
flutter pub get
```
connect to a device with USB debugging on or use Android emulator
```
flutter run
```

### Tech Stack

* Flutter
* Firebase

* Influencer
